# VooFácil

Buscador de passagens aéreas com preços realistas e alerta por WhatsApp.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Digite o código do aeroporto ou nome da cidade (ex: GRU, LIS, JFK, Lisboa, Nova York)
3. Clique em **BUSCAR**

## Tecnologias

- HTML5 + Tailwind CSS
- JavaScript puro
- Totalmente offline (funciona sem internet após carregar)
